keylock-applet
==============

Caps/Num/Scroll Lock indicator applet for X11

# Functionality
- Shows as system tray icon for a single or all keys
- Shows a splash screen if key is active

# Requirements
- C++ compiler (i.e. gcc)
- Qt 4.x
